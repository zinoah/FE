color: 글자 색을 지정하는 속성

        선택자{color: 색상명(영문) | 16진수 숫자 RGB(#fff, #ffffff) | rgb(234, 425, 214) | rgba(255, 255, 255, 1) 
               | hsl(360, 100, 100) | hsla(360, 100, 100, 1) }

        * rhba 에서 a는 투명도( 0: 투명 1: 불투명)
        * hsl(색조, 채도, 밝기)
        * 6개의 색상 지정 방법은 색과 관련된 모든 css 속성에서 사용가능
  • 색상명으로 지정
  • 16진수 rgb로 지정
  • rgb로 지정
  • rgba로 지정
  • hsl로 지정
  • hsla로 지정
